# LiSuify Web

Website for LiSuify

## Environment Variables

| name                       | description                                                    |
| -------------------------- | -------------------------------------------------------------- |
| PUBLIC_NETWORK             | SUI network `mainnet`,`testnet`, or `devnet`.                  |
| PUBLIC_ORIGINAL_LISUIFY_ID | Smart Contract address for original LiSuify                    |
| PUBLIC_LISUIFY_ID          | Smart Contract address for current LiSuify                     |
| PUBLIC_STAKE_POOL_ID       | Smart Contract address for LiSuify pool stake pool information |

## Development mode

`pnpm run dev`

## Build

`pnpm run build`
